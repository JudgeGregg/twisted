Class HiddenUser The server was able to identify the user of this port, but the information was not returned at the request of the user.
Class IdentClient No class docstring; 0/1 instance variable, 0/1 class variable, 1/5 method documented
Class IdentError Can't determine connection owner; reason unknown.
Class IdentServer The Identification Protocol (a.k.a., "ident", a.k.a., "the Ident Protocol") provides a means to determine the identity of a user of a particular TCP connection. Given a TCP port number pair, it returns a character string which identifies the owner of that connection on the server's system.
Class InvalidPort Either the local or foreign port was improperly specified. This should be returned if either or both of the port ids were out of range (TCP port numbers are from 1-65535), negative integers, reals or in any fashion not recognized as a non-negative integer.
Class NoUser The connection specified by the port pair is not currently in use or currently not owned by an identifiable entity.
